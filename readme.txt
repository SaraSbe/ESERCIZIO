ESERCIZIO ABILITÁ INFORMATICHE E TELEMATICHE A.A 2022/2023

- Nel file 'esercizio.sh' è contenuto lo script bash per copiare la cartella 'data' in una cartella con un nome noto, nel caso specifico il Desktop dell'utente.Far girare il programma dalla directory in cui è contenuta la cartella 'data. Si può sostituire la destinazione 'Desktop' con il path che si desidera. 

- Nel file 'Esercizio_parte1_Sbernardori.ipynb' è presente la prima parte dell'esercizio, in cui si lavora su un singolo multipolo di un set di dati. Anche in questo caso far girare il programma dalla directory in cui è contenuta la cartella 'data'. Si può cambiare il set di dati cambiando il numero associato alla variabile 'test' (1,2 o 3).

  Complilare la cella in cui si calcola la matrice di covarianza numerica senza l'ausilio di numpy richiede alcuni minuti (5 minuti nel mio caso). Per velocizzare il processo si consiglia di ridurre il valore della variabile 'measures' a 100.

- Nel file 'Esercizio_parte2_Sbernardori.ipynb' è presente la seconda parte dell'esercizio, in cui si estende la procedura a tre multipoli pari includendo le cross-correlazioni. Far girare il programma dalla directory in cui è contenuta la cartella 'data'. Si può cambiare il set di dati cambiando il numero associato alla variabile 'test' (1,2 o 3).

- Nel file 'figure.pdf' si riportano i plot delle matrici dei tre set di dati della seconda parte dell'esercizio.
